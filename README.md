# CodeQuest

CodeQuest is a gamified programming learning platform that helps users learn coding languages through interactive levels, quizzes, flashcards, and challenges while earning points and progressing through a fun, game-like experience.

## Features
- User authentication (Login & Sign Up)
- Interactive learning levels with coding questions
- Gamified progression system
- Earn points by completing levels and quizzes
- Hint system to help users solve difficult questions
- Flashcards for learning programming concepts and syntax
- Programming quizzes to test knowledge
- Track learning progress and achievements
- Dashboard displaying current progress, points, and completed levels
- Responsive and user-friendly interface

## Project Structure
- `app/login` – User login page
- `app/signup` – User registration page
- `app/dashboard` – Main learning dashboard
- `app/levels` – Interactive coding levels and challenges
- `app/quizzes` – Programming quizzes
- `app/flashcards` – Flashcard learning section
- `app/profile` – User profile and progress
- `app/api` – Backend API routes
- `app/components` – Reusable UI components
- `app/utils` – Helper functions and utilities

---

## How It Works

1. Users create an account or log in.
2. Choose a programming language to learn.
3. Complete interactive levels containing coding questions and challenges.
4. Use hints whenever a question becomes difficult.
5. Earn points for correct answers and completing levels.
6. Study programming concepts using flashcards.
7. Reinforce learning with quizzes.
8. Track progress and unlock new levels as skills improve.

## Goal

CodeQuest aims to make learning programming engaging, interactive, and enjoyable by combining educational content with game mechanics that motivate users to keep learning.
